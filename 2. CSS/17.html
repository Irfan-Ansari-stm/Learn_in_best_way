15.1 CSS Filters

CSS filters apply graphical effects like blurring or color shifting to elements. They're powerful tools for creating mood, focus, and visual hierarchy.

blur(), brightness(), contrast()

Definition: Basic filters that modify the appearance of an element through pixel manipulation.

Detail:

  • blur(radius): Applies Gaussian blur - higher values = more blur
  • brightness(amount): Multiplies pixel values - 0% = black, 100% = original, >100% = brighter
  • contrast(amount): Adjusts difference between light and dark areas - 0% = gray, 100% = original, >100% = higher contrast

Syntax:

filter: blur(5px);
filter: brightness(1.5);
filter: contrast(180%);
filter: blur(3px) brightness(1.2) contrast(1.3);

Example:

Blurred & Enhanced
.card {
  filter: blur(2px);
}

.card:hover {
  filter: blur(0px);
}

Uses: Creating depth (blurred backgrounds), subtle highlights (brightened buttons), and visual emphasis (high-contrast elements).

ðŸ’¡ Pro Tip: Use backdrop-filter: blur() for frosted glass effects on containers behind other elements!

saturate(), hue-rotate(), invert()

Definition: Color-manipulating filters that alter the chromatic properties of elements.

Detail:

  • saturate(amount): Controls color intensity - 0% = grayscale, 100% = normal, >100% = oversaturated
  • hue-rotate(angle): Rotates colors around the color wheel - 0deg = no change, 360deg = full circle
  • invert(amount): Inverts pixel values - 0% = unchanged, 100% = complete negative

Syntax:

filter: saturate(200%);
filter: hue-rotate(180deg);
filter: invert(100%);
filter: saturate(150%) hue-rotate(90deg) invert(20%);

Example:

Vibrant & Hue-shifted
.theme-toggle {
  filter: hue-rotate(180deg);
}

.theme-toggle.active {
  filter: hue-rotate(0deg);
}

Uses: Dark/light mode toggles, color theme variations, artistic effects, and accessibility adjustments.

ðŸ’¡ Pro Tip: Combine hue-rotate(180deg) with invert(100%) for instant dark mode conversion!

drop-shadow() vs box-shadow

Definition: Two different ways to create shadows with distinct behaviors.

Detail:

  • box-shadow: Rectangular shadow based on element's bounding box
  • drop-shadow(): Pixel-perfect shadow following actual shape (including transparency)

Syntax:

/* box-shadow */
box-shadow: 10px 10px 15px rgba(0,0,0,0.3);

/* drop-shadow */
filter: drop-shadow(10px 10px 15px rgba(0,0,0,0.3));

Example:

Box Shadow
Drop Shadow

Key Differences:

Feature box-shadow drop-shadow()
Shape Rectangular Follows actual pixels
Transparency Ignored Honored
Performance Faster Slower (pixel processing)
Blur Control Yes Yes
Multiple Shadows Yes Yes

Uses: box-shadow for simple UI elements, drop-shadow() for icons, logos, and irregular shapes.

ðŸ’¡ Pro Tip: Use drop-shadow() for any element with rounded corners or transparency - it looks infinitely more professional.

15.2 Blend Modes and Compositing

Blend modes control how overlapping elements interact visually. They're inspired by Photoshop layer blending and create stunning composite effects.

mix-blend-mode property

Definition: Determines how an element blends with its background and underlying content.

Detail: Common values:

  • normal - default (no blending)
  • multiply - darkens image (like ink on paper)
  • screen - lightens image (like slide projector)
  • overlay - combines multiply and screen
  • darken - takes darker pixels from both
  • lighten - takes lighter pixels from both
  • color-dodge - brightens base color
  • color-burn - darkens base color

Syntax:

.layer {
  mix-blend-mode: multiply;
}

Example:

Screen
Multiply
.blend-layer:nth-child(1) {
  mix-blend-mode: screen;
}

.blend-layer:nth-child(2) {
  mix-blend-mode: multiply;
}

Uses: Creating luminous text effects, layered graphics, artistic overlays, and modern web design aesthetics.

ðŸ’¡ Pro Tip: Always set a background color on the container when using mix-blend-mode â€” otherwise it blends with the page!

background-blend-mode

Definition: Blends multiple background images or colors within a single element.

Detail: Works with multiple background layers defined with comma-separated values.

Syntax:

.element {
  background: 
    linear-gradient(45deg, #00f5ff, #ffb74d),
    url('pattern.png');
  background-blend-mode: overlay;
}

Example:

Gradient Overlay 
.hero-section {
  background: 
    linear-gradient(rgba(0, 0, 0, 0.7), rgba(0, 0, 0, 0.7)),
    url('background.jpg');
  background-blend-mode: overlay;
  color: white;
}

Uses: Text over images with readable contrast, modern hero sections, and dynamic gradient effects.

ðŸ’¡ Pro Tip: Combine background-blend-mode: overlay with a semi-transparent gradient to make text pop over photos!

Isolation and stacking contexts

Definition: How elements are organized in the rendering stack and how blend modes interact with their surroundings.

Detail:

  • Stacking context: A group of elements that render as a single unit
  • isolation: isolate; Creates a new stacking context, preventing blend modes from affecting parent elements

Syntax:

.container {
  isolation: isolate;
}

.parent {
  mix-blend-mode: multiply;
}

.child {
  mix-blend-mode: screen;
  isolation: isolate; /* Isolates this child from parent's blend mode */
}

Example:

Parent Blend
Isolated Child

Uses: Complex UIs with nested blended elements, avoiding unintended visual artifacts, and precise control over composition.

ðŸ’¡ Pro Tip: Use isolation: isolate; whenever you want a blended element to behave independently from its parent's blending.

Definition: Advanced applications of blend modes for artistic and functional effects.

Detail: Creative techniques include:

  • Text glow: mix-blend-mode: screen on text with colored background
  • Color grading: Layer colored overlays with multiply or overlay to adjust image tone
  • Texture overlays: Add noise textures with overlay for film-like quality
  • Interactive color shifts: Animate hue-rotate() + mix-blend-mode for dynamic interfaces

15.3 CSS Masks and Clipping

Masks and clipping allow you to reveal or hide portions of an element based on shapes or images â€” perfect for creative layouts and image treatments.

mask-image and mask-size

Definition: mask-image uses an image or gradient to define what parts of an element are visible. mask-size controls its dimensions.

Detail:

  • Black areas = invisible
  • White areas = fully visible
  • Grayscale = partial transparency
  • Alpha channel = opacity control

Syntax:

.element {
  mask-image: url('mask.svg');
  mask-size: cover;
  mask-repeat: no-repeat;
  mask-position: center;
}

Example:

Masked Gradient 
.profile-photo {
  mask-image: radial-gradient(circle at center, black 70%, transparent 100%);
  mask-size: cover;
  mask-repeat: no-repeat;
  background: url('photo.jpg') center/cover;
}

Uses: Circular profile pictures, fade-out gradients, custom-shaped containers, and image vignettes.

ðŸ’¡ Pro Tip: Use SVG masks for precision control â€” they scale perfectly and support animation!

mask-position and mask-repeat

Definition: Control positioning and repetition of mask images, similar to background properties.

Detail: These properties work exactly like their background counterparts:

  • mask-position: Where the mask starts (center, top left, 50% 30%)
  • mask-repeat: Whether to repeat (repeat, no-repeat, round, space)

Syntax:

.element {
  mask-image: url('pattern-mask.svg');
  mask-position: center;
  mask-repeat: repeat-x;
  mask-size: 200% 100%;
}

Example:

Repeating Mask Pattern 
.striped-effect {
  mask-image: repeating-linear-gradient(
    to right,
    black, black 10px,
    transparent 10px, transparent 20px
  );
  mask-size: 40px 100%;
  background: linear-gradient(135deg, #00f5ff, #ffb74d);
}

Uses: Stripe patterns, window blinds effects, textured reveals, and animated mask transitions.

ðŸ’¡ Pro Tip: Combine mask-position with animation to create sliding reveal effects!

clip-path for complex shapes

Definition: clip-path cuts out a specific shape from an element, revealing only the portion inside the path.

Detail: Supports:

  • Basic shapes: circle(), ellipse(), polygon(), inset()
  • SVG paths
  • URL references to SVG clip paths

Syntax:

.element {
  clip-path: circle(50% at center);
  clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
  clip-path: inset(20px round 10px);
}

Example:

Star Shape
.hexagon {
  clip-path: polygon(30% 0%, 70% 0%, 100% 50%, 70% 100%, 30% 100%, 0% 50%);
  background: linear-gradient(135deg, #00f5ff, #ffb74d);
  width: 200px;
  height: 200px;
}

Uses: Custom shaped cards, interactive UI elements, product showcases, and modern web design.

ðŸ’¡ Pro Tip: Use clip-path: inset(10px round 15px); for easy rounded rectangles without border-radius!

Detail: Key differences:
