6.1 Color Theory for CSS

Color isn't just decoration — it's communication. Learn how to use it intentionally.

Contrast Principles:

  • High contrast: Black text on white background — maximum readability
  • Medium contrast: Dark gray on light gray — subtle but readable
  • Low contrast: Light gray on white — poor readability

Pro Tip: Use the 60-30-10 rule: 60% dominant color, 30% secondary color, 10% accent color.

Tools for Checking Contrast:

  • WebAIM Contrast Checker
  • Contrast Ratio
  • Browser DevTools (Accessibility panel)
  • Color Oracle (color blindness simulator)

Best Practices:

  1. Always test contrast ratios before deployment
  2. Use dark text on light backgrounds (better readability)
  3. Avoid pure red/green combinations for critical information
  4. Don't rely on color alone to convey meaning — add icons or text labels
  5. Test with grayscale filters to check if information is still clear

6.2 Background Properties

Control every aspect of your element's background — from simple colors to complex images.

Background color and images

Set the foundation of your background with color and/or image.


/* Simple background color */
.simple-bg {
    background-color: #0a192f;
}

/* Background image */
.image-bg {
    background-image: url('pattern.png');
}

/* Multiple properties in shorthand */
.shorthand {
    background: #0a192f url('hero.jpg') no-repeat center center / cover;
}

/* Background with fallback */
.fallback-bg {
    background-color: #0a192f; /* Fallback for old browsers */
    background-image: url('modern-pattern.webp');
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
}
Solid Color Background
Pattern Background (SVG)

Best Practices:

  • Always provide a background-color fallback
  • Use SVG for simple patterns (small file size)
  • Compress images for performance
  • Use WebP format for better compression
  • Consider lazy loading for hero images
Background size and position

Control how your background image scales and positions within its container.


/* Background size values */
.size-contain {
    background-size: contain; /* Scale to fit entirely */
}

.size-cover {
    background-size: cover; /* Cover entire area (cropped) */
}

.size-100pct {
    background-size: 100% 100%; /* Stretch to fill exactly */
}

.size-auto {
    background-size: auto; /* Original size */
}

.size-custom {
    background-size: 200px 150px; /* Specific dimensions */
}

/* Background position values */
.position-center {
    background-position: center; /* Centered */
}

.position-top {
    background-position: top; /* Top center */
}

.position-bottom-right {
    background-position: bottom right; /* Bottom right corner */
}

.position-50pct {
    background-position: 50% 50%; /* Exactly center */
}

.position-offset {
    background-position: 20px 30px; /* Offset from top-left */
}

/* Shorthand combination */
.shorthand-bg {
    background: url('image.jpg') center / cover no-repeat;
}
Background-size: contain (fits entirely)
Background-size: cover (fills completely)

When to use each:

  • contain: When you need to see the entire image (logos, illustrations)
  • cover: When you want the background to fill the space (hero sections, banners)
  • 100% 100%: Rarely used — distorts images
  • auto: Default — original image size

Positioning Tips:

  • Use center for balanced compositions
  • Use top center for headers
  • Use bottom center for footers
  • Use pixel offsets for precise alignment

Control how background images tile and whether they scroll with content.

Multiple backgrounds

Stack multiple background images and colors on a single element.


/* Multiple backgrounds separated by commas */
.multi-bg {
    background-image: 
        url('pattern.png'),
        url('texture.jpg'),
        linear-gradient(to right, rgba(0, 209, 255, 0.1), transparent);
    
    background-position: 
        top left,
        center center,
        top left;
    
    background-repeat: 
        repeat,
        no-repeat,
        no-repeat;
    
    background-size: 
        100px 100px,
        cover,
        100% 100%;
}

/* Shorthand version (order must match) */
.shorthand-multi {
    background: 
        url('pattern.png') top left / 100px 100px repeat,
        url('texture.jpg') center / cover no-repeat,
        linear-gradient(to right, rgba(0, 209, 255, 0.1), transparent) top left / 100% 100% no-repeat;
}

/* Background color as part of stack */
.bg-with-color {
    background: 
        #0a192f, /* Bottom layer */
        url('overlay.png') center / contain no-repeat; /* Top layer */
}
Multiple Backgrounds: Cyan dots + Amber dots

Pro Tips:

  • Order matters — first item is top layer, last is bottom
  • Use gradients for subtle overlays instead of images
  • Combine with background-blend-mode for advanced effects
  • Limit to 3-4 layers for performance

6.3 Gradients

Create smooth transitions between colors — the most powerful background tool in modern CSS.

Linear gradients

Color transitions along a straight line.


/* Basic linear gradient */
.linear-basic {
    background: linear-gradient(to right, #00d1ff, #00a8ff);
}

/* Direction variations */
.linear-top {
    background: linear-gradient(to top, #ffa726, #ff4081);
}

.linear-diagonal {
    background: linear-gradient(45deg, #00d1ff, #00a8ff);
}

.linear-corner {
    background: linear-gradient(to bottom right, #00d1ff, #00a8ff);
}

/* Multiple color stops */
.multi-stop {
    background: linear-gradient(to right, 
        #00d1ff, 
        #00a8ff, 
        #007fff, 
        #0052ff);
}

/* Precise positioning */
.precise-stops {
    background: linear-gradient(to right, 
        #00d1ff 0%, 
        #00a8ff 25%, 
        #007fff 50%, 
        #0052ff 75%, 
        #002bff 100%);
}

/* Transparent gradients */
.transparent {
    background: linear-gradient(to right, 
        rgba(0, 209, 255, 0.8), 
        transparent);
}

/* Text gradients (requires background-clip) */
.text-gradient {
    background: linear-gradient(to right, #00d1ff, #00a8ff);
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
}
Linear Gradient: Cyan to Dark Cyan
Diagonal Gradient: Amber to Pink
Multi-Stop Gradient

Direction Syntax:

  • to right, to left, to top, to bottom
  • 0deg = right, 90deg = top, 180deg = left, 270deg = bottom
  • 45deg = top-right diagonal

Pro Tip: Use transparent in gradients for subtle overlays and depth effects.

Radial gradients

Color transitions radiating outward from a central point.


/* Basic radial gradient */
.radial-basic {
    background: radial-gradient(circle, #00d1ff, #00a8ff);
}

/* Shape variations */
.radial-circle {
    background: radial-gradient(circle, #ffa726, #ff4081);
}

.radial-ellipse {
    background: radial-gradient(ellipse, #00d1ff, #00a8ff);
}

/* Position control */
.radial-position {
    background: radial-gradient(circle at top left, #00d1ff, #00a8ff);
}

.radial-center {
    background: radial-gradient(circle at center, #00d1ff, #00a8ff);
}

/* Size control */
.radial-closest-side {
    background: radial-gradient(closest-side, #00d1ff, #00a8ff);
}

.radial-farthest-corner {
    background: radial-gradient(farthest-corner, #00d1ff, #00a8ff);
}

/* Multiple color stops */
.radial-multi {
    background: radial-gradient(circle, 
        #00d1ff 0%, 
        #00a8ff 30%, 
        #007fff 60%, 
        #0052ff 100%);
}

/* Inset radial (inner glow) */
.inset-glow {
    background: radial-gradient(circle at center, 
        rgba(0, 209, 255, 0.3) 0%, 
        transparent 70%);
}

Conic gradients

Color transitions arranged around a central point like a pie chart.


/* Basic conic gradient */
.conic-basic {
    background: conic-gradient(from 0deg, #00d1ff, #00a8ff, #007fff, #0052ff);

/* Clock face effect */
.clock-face {
    background: conic-gradient(
        from 0deg,
        #00d1ff 0deg 90deg,
        #00a8ff 90deg 180deg,
        #007fff 180deg 270deg,
        #0052ff 270deg 360deg
    );
}

/* Color wheel */
.color-wheel {
    background: conic-gradient(
        red, orange, yellow, green, cyan, blue, purple, red
    );
}

/* Semi-circle */
.semi-circle {
    background: conic-gradient(
        from 0deg,
        #00d1ff 0deg 180deg,
        transparent 180deg 360deg
    );
}

/* Gradient ring */
.gradient-ring {
    background: conic-gradient(
        from 0deg,
        #00d1ff 0deg 10deg,
        transparent 10deg 350deg,

/* Modern progress indicator */
.progress-indicator {
    background: conic-gradient(
        #00d1ff 0% 75%,

Background clipping and origins

Control where backgrounds appear within an element's box model.


/* Background origin: where background is positioned */
.origin-padding {
    background-origin: padding-box; /* Default */
}

.origin-border {
    background-origin: border-box;
}

.origin-content {
    background-origin: content-box;
}

/* Background clip: where background is painted */
.clip-border {
    background-clip: border-box; /* Default */
}

.clip-padding {
    background-clip: padding-box;
}

.clip-content {
    background-clip: content-box;
}

/* Text clipping (for gradient text) */
.gradient-text {
    background: linear-gradient(to right, #00d1ff, #00a8ff);
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
}

/* Border-only background */
.border-bg {
    background: #00d1ff;
    background-clip: padding-box;
    border: 5px solid #0a192f;
    border-radius: 10px;
}

/* Double border effect */
.double-border {
    background: linear-gradient(to right, #00d1ff, #00a8ff);
    background-clip: content-box;
    border: 3px solid transparent;
    border-radius: 10px;
    position: relative;
}

.double-border::before {
    content: '';
    position: absolute;
    top: -3px;
    left: -3px;
    right: -3px;
    bottom: -3px;
    background: linear-gradient(45deg, #00d1ff, #00a8ff);
    border-radius: 13px;
    z-index: -1;
}
Background-clip: content-box (Only inside padding)
Background-clip: border-box (Includes border)
Background-clip: padding-box (Inside padding, outside border)

Key Differences:

  • background-origin: Where the background image starts
  • background-clip: Where the background is painted

Pro Tip: Combine background-clip: text with color: transparent for stunning gradient text effects.

Blend Mode Reference:

Mode Effect Best For
normal No blending Default
multiply Darkens — multiplies colors Shadows, depth
screen Lightens — inverts multiply Highlights, glow
overlay Mix of multiply and screen Texture, detail
soft-light Subtle lighting Subtle enhancements
hard-light Strong lighting Dramatic effects
difference Inverts base color Abstract art

backdrop-filter effects

Create frosted glass and blur effects behind elements.


/* Frosted glass effect */
.frosted-glass {
    backdrop-filter: blur(10px);
    -webkit-backdrop-filter: blur(10px); /* Safari support */
    background-color: rgba(255, 255, 255, 0.1);
    border: 1px solid rgba(255, 255, 255, 0.2);
    border-radius: 16px;
}

/* Multiple backdrop filters */
.advanced-frosted {
    backdrop-filter: blur(15px) saturate(180%);
    -webkit-backdrop-filter: blur(15px) saturate(180%);
    background-color: rgba(255, 255, 255, 0.05);
    border: 1px solid rgba(255, 255, 255, 0.1);
}

/* With contrast enhancement */
.high-contrast-frosted {
    backdrop-filter: blur(10px) contrast(120%);
    -webkit-backdrop-filter: blur(10px) contrast(120%);
    background-color: rgba(255, 255, 255, 0.1);
}

/* Combined with border */
.modal-overlay {
    backdrop-filter: blur(8px);
    -webkit-backdrop-filter: blur(8px);
    background-color: rgba(10, 25, 47, 0.8);
    border: 1px solid rgba(0, 209, 255, 0.2);
    border-radius: 12px;
}

/* Modern modal design */
.modal {
    position: fixed;
    top: 50%;
    left: 50%;

    -webkit-backdrop-filter: blur(15px);
    background-color: rgba(255, 255, 255, 0.05);
    border: 1px solid rgba(0, 209, 255, 0.2);
    border-radius: 20px;

}
Frosted Glass Effect

Backdrop Filter Values:

  • blur(): Creates Gaussian blur
  • brightness(): Adjusts brightness
  • contrast(): Adjusts contrast
  • grayscale(): Converts to grayscale
  • hue-rotate(): Rotates hue
  • invert(): Inverts colors
  • saturate(): Increases saturation
  • sepia(): Applies sepia tone

Combining Filters:


.backdrop-combo {
    backdrop-filter: blur(10px) brightness(1.2) contrast(1.1);
}

Browser Support: Excellent in modern browsers (Chrome 85+, Firefox 103+, Safari 15+).

Performance Tip: Avoid applying backdrop-filter to large areas — it's computationally expensive.

Modern Use Case: Modal dialogs, floating cards, navigation bars with blurred backgrounds.
