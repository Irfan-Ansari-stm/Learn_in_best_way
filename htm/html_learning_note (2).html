1.2 Setting Up Your Development Environment

This section explains everything needed to start coding HTML professionally — from editors to file structure to creating your first HTML document.

Text Editors and IDEs (VS Code, Sublime, Atom)

Executive Summary

To write HTML, you need a text editor or IDE. VS Code is the most widely used due to extensions, performance, and built‑in tools.

Deep Explanation

VS Code: Free, fast, intelligent auto‑complete, Emmet support, Live Server, Extensions.

Sublime Text: Extremely lightweight, very fast, great for quick editing.

Atom: Clean interface, open‑source, but slower and less maintained today.

Example: Using VS Code Effectively 
<!-- Useful VS Code Features -->
Emmet: type "!" + Enter → full HTML boilerplate
Extensions: Live Server, Prettier, Material Icons
Ctrl + B → Toggle file explorer
Ctrl + ` → Open terminal

Browser Developer Tools

Executive Summary

Every modern browser includes built‑in tools to inspect HTML, edit CSS live, debug JavaScript, and analyze page performance.

Deep Explanation
  • Elements panel: View & edit DOM structure.
  • Console: Run JavaScript commands.
  • Network: View page load behavior.
  • Performance: Analyze rendering speed.
Example: Inspecting HTML 
Right‑click → Inspect
Elements tab → Shows real-time HTML
Edit text, modify CSS → Browser updates instantly

File Structure and Organization

Executive Summary

A clean folder structure keeps your project scalable, readable, and easy to maintain.

Recommended Folder Layout 
project-folder/
│
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── assets/
    ├── images/
    └── icons/
Why Structure Matters
  • Makes large projects manageable
  • Browser finds files easily
  • Improves teamwork & version control

Creating Your First HTML File

Executive Summary

Creating an HTML file is simple — write code in a text editor, save as .html, and open in a browser.

Step-by-Step Guide
  1. Create a folder named my-first-website
  2. Open folder in VS Code
  3. Create index.html
  4. Write the following code:
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello Irfan!</h1>
    <p>Your first HTML page is working.</p>
  </body>
</html>
