1.4 The HTML Document Template

This chapter teaches you the core structure every HTML document must follow.

DOCTYPE Declaration

Executive Summary

The DOCTYPE declaration tells the browser which version of HTML the document uses. Modern HTML always uses: 

<!DOCTYPE html>
Deep Explanation

In older HTML versions (HTML 4.01, XHTML), the DOCTYPE was extremely long. HTML5 simplified everything to just: 

<!DOCTYPE html>

This enables:

  • Standards Mode (correct rendering)
  • Modern HTML5 features
  • Consistent browser behavior

<html> Element & Language Attribute

Executive Summary

The <html> tag is the root of every HTML document. The lang attribute improves accessibility and SEO.

Syntax 
<html lang="en">
...
</html>

Common values:

  • en ‚Äî English
  • hi ‚Äî Hindi
  • ar ‚Äî Arabic
Real Use Case

Screen readers rely on the lang attribute to choose correct pronunciation patterns.

<head> Section Essentials

Executive Summary

The <head> contains important metadata: character encoding, title, styles, scripts, SEO tags.

Common Head Structure 
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Webpage</title>
</head>
Deep Explanation
  • <meta charset="UTF-8"> ‚Äî ensures correct character display
  • viewport ‚Äî enables responsive design
  • <title> ‚Äî shown in browser tab
  • Styles & scripts (CSS, JavaScript)

<body> Element

Executive Summary

Everything visible on the page goes inside <body>.

Example 
<body>
  <h1>Welcome!</h1>
  <p>This is visible on the webpage.</p>
</body>
